# Dex Sovereign: Neural Training Lab

This Space is the dedicated training ground for the Dex Sovereign Routing Matrix.
It fine-tunes ModernBERT-Base to handle a balanced 9-intent decision space.

## Architecture Status
- Core Model: ModernBERT-Base
- Neural Pathways: 9 (Routing Matrix)
- DNA Volume: ~1000 Balanced Samples

## Operations
1. Ensure intent_dataset.csv is synced from the Linode.
2. Click START BAKE to initiate backpropagation.
3. Once the Loss plateaus, pull the weights back to the local Gatekeeper.
