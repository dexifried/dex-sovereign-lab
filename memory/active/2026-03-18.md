## Hardware Constraint — DEX Local Models
- Local models (qwen-sub, hermes3, qwen-4b, qwen-lite, deepseek-dex-coder) all run on a **single RTX 3060 12GB**
- Running multiple local agents in parallel causes VRAM spillover into system RAM → slow/broken responses (hermes3 returned empty in parallel test)
- **Rule: spawn local DEX agents serially, not in parallel.** Max 1 local model at a time.
- Cloud agents (coder/GPT, 5-4codex, anthropic) are fine to parallelize — they run off-device.
- Mixed parallel (1 local + 1 cloud) is probably fine; avoid 2+ local at once.
